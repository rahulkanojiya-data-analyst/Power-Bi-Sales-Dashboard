# Sales Performance Dashboard | Power BI

## Project Overview

This project presents an interactive Sales Performance Dashboard built using Power BI. The dashboard helps analyze sales trends, profitability, regional performance, and category-wise performance to support business decision-making.

## Tools Used

* Power BI
* Power Query
* DAX
* Excel Dataset

## Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Profit Margin %

## Dashboard Features

* Sales by Category Analysis
* Sales by Region Analysis
* Monthly Sales Trend
* Top Performing Products
* Interactive Filters and Slicers
* Business Performance Monitoring

## Data Preparation

* Removed duplicate records
* Checked and corrected data types
* Handled missing values
* Created calculated measures using DAX
* Prepared data model for analysis

## Key Insights

* Technology category generated the highest sales.
* West region contributed the largest share of revenue.
* Furniture category showed comparatively lower profitability.
* Sales increased significantly during peak seasonal periods.
* A small group of products contributed a large portion of total sales.
